CubeSwipe

모바일 사이트에서 많이 사용되어지는 플리킹 플러그인을 응용하여 만들어진 원형효과의 플러그인 입니다. 자바스크립트와 jQuery 동시에 지원하며 슬라이더의 무한 루프와 페이징 기능등 많이 사용되어지는 기능을 넣었습니다.

(IE7, IE8 등 css3를 지원하지 않는 브라우저에서는 사용할 수 없습니다.)

How to Use

Method

toPrev 이전 슬라이더 이동
toNext 다음 슬라이더 이동
toSlide(index) 지정된 슬라이더 이동
getPos 현재 슬라이더 인덱스
startSlideShow 슬라이더쇼 시작
stopSlideShow 슬라이더쇼 정지

jQuery options

$list Type: jQuery Selector (default: $( this ).find( 'ul li' ))
리스트
$pages Type: jQuery Selector (default: null)

$toStart Type: jQuery Selector (default: null)
슬라이드쇼 시작 버튼
$toStop Type: jQuery Selector (default: null)
슬라이드쇼 멈춤 버튼
$toPrev Type: jQuery Selector (default: null)
이전 이동 버튼
$toNext Type: jQuery Selector (default: null)
다음 이동 버튼

Common options

startEvents Type: Array (default: ['click', 'mouseover'])
toStart element 이벤트
stopEvents Type: Array (default: ['click', 'mouseover'])
toStop element 이벤트
moveEvents Type: Array (default: ['click', 'mouseover'])
toPrev and toNext element 이벤트
pageEvents Type: Array (default: ['click', 'mouseover'])
pages element 이벤트
touchMinumRange Type: Integer (default:10)
사용자 터치시, 다음(혹은 이전) 슬라이더로 넘어갈 기준값(백분율)
duration Type: Integer (default: 400)
애니메이션 시간
loop Type: Boolean (default: true)
루프 여부. false로 설정되면 마지막(처음) 슬라이더에서 처음(마지막) 슬라이더로 이동 안함
slideShowTime Type: Boolean or Integer (default: 3000)
슬라이더쇼 시간
create Type: Function
생성시 콜백 함수
active Type: Function
슬라이더 이동 후 콜백 함수
